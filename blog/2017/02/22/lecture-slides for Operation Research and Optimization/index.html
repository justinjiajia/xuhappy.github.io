Project 分组情况

  • SVM for Handwritten Digits Classification. Dataset is available on THE MNIST DATABASE. In this project, you need to:
    1. Apply the basic SVM model to train the training set and classify the digits in the test set.
    2. Use kernel method to retrain the SVM model and classify the digits.
  • K-means for image clustering. Dataset is available on THE MNIST DATABASE. In this project, you need to:
    1. Combine both the training set and test set into one set and randomly pick 50000 images.
    2. Use K-means to cluster the images you have chosen in Step 1 into 10 different classes and compute the error rate using the ground truth.
    3. Apply Fuzz C-means method in C-means to cluster the images you have chosen in Step 1 into 10 different classes and compute the error rate using the ground truth. Compare the result with that of Step 2.
